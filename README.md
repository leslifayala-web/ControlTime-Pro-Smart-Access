# ControlTime-Pro-Smart-Access
Proyecto de control de acceso y asistencia de maestros utilizando Arduino.

## Proyecto Integrador
**Instituto Católico Karol Wojtyla**

### Bachillerato Técnico Vocacional en Desarrollo de Software

## Descripción

ControlTime Pro Smart Access es un sistema inteligente desarrollado para controlar el acceso y registrar la asistencia de los maestros mediante el uso de Arduino y diferentes componentes electrónicos.

Este proyecto es para la Feria de Microcontroladores ICKW 2026.

## Objetivo

Desarrollar un sistema automatizado que permita controlar el ingreso de los maestros de forma segura, rápida y eficiente utilizando tecnología basada en Arduino.
