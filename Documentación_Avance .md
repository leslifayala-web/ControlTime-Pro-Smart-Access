# Objetivo del proyecto

Desarrollar un sistema inteligente que permita controlar el acceso y registrar la asistencia de los maestros mediante el uso de Arduino y diferentes componentes electrónicos, brindando un proceso más seguro, rápido y automatizado.

# Desarrollo del circuito

El circuito fue diseñado en la plataforma Tinkercad utilizando un Arduino Uno como controlador principal. Se conectaron los sensores y actuadores necesarios para simular el funcionamiento del sistema de acceso y asistencia de los maestros.

Durante el diseño se verificó que todas las conexiones estuvieran correctamente realizadas para garantizar el funcionamiento de cada componente.

# Programación

La programación fue desarrollada en lenguaje C++ para Arduino.

El código incluye:

- Declaración de variables.
- Configuración de entradas y salidas.
- Función `setup()`.
- Función `loop()`.
- Lectura de sensores.
- Control de actuadores.

# Pruebas realizadas

Se realizaron diferentes pruebas para verificar el correcto funcionamiento del sistema.

Entre ellas:

- Comprobación de la lectura de los sensores.
- Verificación del funcionamiento de los actuadores.
- Pruebas de activación y desactivación del sistema.
- Revisión de las conexiones del circuito.
- Validación de la lógica del programa.

# Problemas encontrados

Durante el desarrollo se presentaron algunos inconvenientes relacionados con las conexiones del circuito y con la programación de algunos componentes, lo que provocó que el sistema no respondiera correctamente en las primeras pruebas.

# Soluciones implementadas

Para solucionar estos inconvenientes se revisó el cableado del circuito, se corrigieron errores en el código y se realizaron nuevas pruebas hasta lograr el funcionamiento esperado de la simulación.

# Resultado obtenido

Se obtuvo una simulación funcional en Tinkercad que permite comprobar el correcto funcionamiento del sistema antes de construir el prototipo físico.
